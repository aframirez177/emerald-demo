# Walkthrough script — 60–90 seconds

For screen recording on desktop (1440×900 viewport, Chrome, hide bookmarks bar) + a separate 30s vertical clip on real iPhone for the sales deck's mobile slide.

## Before recording

- Open https://aframirez177.github.io/emerald-demo/ in a fresh incognito window so the language detector routes to `/fr/` (assuming macOS language = French or add `?lang=fr` to force).
- Zoom 100%, no DevTools, cursor visible, no notifications.
- Use QuickTime or Loom — prefer 1080p.

## Mobile clip (~30s, vertical)

iPhone 13/14, Safari, portrait. Incognito.

1. Land on `/` → tap English.
2. Scroll through hero slowly (stops at natural breakpoints).
3. Hit "Explore the collection" CTA.
4. Tap a piece → show the form on mobile (do not submit).
5. End on the footer with the demo disclosure visible.

## Don'ts

- Don't submit the contact form in the video (Web3Forms free tier has a submission limit — save them for real prospects).
- Don't zoom or crop mid-recording; do it in post.
- Don't record with autoplay audio notifications firing.
